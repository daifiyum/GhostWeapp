# GhostWeapp
> 一个简单的Ghost微信小程序，基于Ghost api

### 功能

支持搜索（Ghost没有提供搜索api，但参考Ghost目前支持的搜索实现，使用文章列表api，对10000条数据进行模糊搜索）

首页轮播图（ghost后台将文章标记为featured即可）

分类（基于ghost标签）

文章收藏（无需登录，使用本地存储数据）

### 用到的库

vant weapp

mp-html
